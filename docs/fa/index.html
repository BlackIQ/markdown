آموزش Markdown.

مارک‌داون عنوان یک زبان قالب‌بندی متن و نیز یک ابزار تبدیل متن به اچ‌تی‌ام‌ال است که اجازه می‌دهد کاربران متن را به صورت ساده وارد کنند و سپس به صورت خودکار آن را به متن غنی تبدیل نمایند.

اینجا درباره این زبان خیلی چیزا یاد میگیرم و یاد میگیریم چطوری ازش استفاده کنیم.

استایل متنی

منظور از استایل های متنی همان کج کردن و ضخیم کردن متن میباشد.

در HTML از b تگ ها برای ضخیم کردن استفاده میشد. در اینجا از ** در ابتدای متن و انتهای متن استفاده میشود.

HTML

بنده <b>امیر</b> هستم. از آشنایی با <b>شما</b> خوشبختم.

خروجی

بنده امیر هستم. از آشنایی با شما خوشبختم.


Markdown

بنده **امیر** هستم. از آشنایی با **شما** خوشبختم.

خروجی

بنده امیر هستم. از آشنایی با شما خوشبختم.



اما چطوری متن را ایتالیک کنیم. در HTML از تگ i استفاده میشود. اما در اینجا فقط یک ستاره استفاده میشود.

HTML

خروجی

اون گفت ما عاشق استفاده از گیت هستم.


Markdown

اون گفت *ما عاشق استفاده از گیت هستم*.

خروجی

اون گفت ما عاشق استفاده از گیت هستم.
