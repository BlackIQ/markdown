استایل متنی

منظور از استایل های متنی همان کج کردن و ضخیم کردن متن میباشد.

In HTML we use b tags. In md we just use ** in start and end of the text. Look at examples.

HTML

This is <b>Amir</b>. Nice to meet <b>you</b>.

خروجی

This is Amir. Nice to meet you.


Markdown

This is **Amir**. Nice to meet **you**.

خروجی

This is Amir. Nice to meet you.



Now, this is how to make a text italic. In HTML people use i and in Markdown we use stars. Not 2. Just one!

HTML

He said <i>I love to use Git in my projects</i>.

خروجی

He said I love to use Git in my projects.


Markdown

He said *I love to use Git in my projects*.

خروجی

He said I love to use Git in my projects.


This was italic and bold methods!
