Learn Markdown.

Markdown is a lightweight markup language for creating formatted text using a plain-text editor. John Gruber and Aaron Swartz created Markdown in 2004 as a markup language that is appealing to human readers in its source code form.

Here you will learn about it more and how to use that.


Headings

What are Headings? The title or the name of a document or even an chapter of a book is a Heading. Now, let's learn how to use them!

In HTML we use H tags. These are Headings. But not md. We use # instead of H tags. Fitst I will show you HTML way and next, Markdown!

HTML

<h1>Heading</h1> <h2>Heading</h2> <h3>Heading</h3> <h4>Heading</h4> <h5>Heading</h5> <h6>Heading</h6>

Output

Heading

Heading

Heading

Heading

Heading
Heading

Ok, right. Let's go for markdown.

Markdown

# Heading ## Heading ### Heading #### Heading ##### Heading ###### Heading

Output

Heading

Heading

Heading

Heading

Heading
Heading

As you can see, we use #. If your heading is H2, use ##. Hope you got the point!


Text style

The meaning of text styles is the same as bolding and italicizing the text.

In HTML we use b tags. In md we just use ** in start and end of the text. Look at examples.

HTML

This is <b>Amir</b>. Nice to meet <b>you</b>.

Output

This is Amir. Nice to meet you.


Markdown

This is **Amir**. Nice to meet **you**.

Output

This is Amir. Nice to meet you.



Now, this is how to make a text italic. In HTML people use i and in Markdown we use stars. Not 2. Just one!

HTML

He said <i>I love to use Git in my projects</i>.

Output

He said I love to use Git in my projects.


Markdown

He said *I love to use Git in my projects*.

Output

He said I love to use Git in my projects.


This was italic and bold methods!


Links

You may wanted to add a link in your text. You wanna use HTML tag!? Stop doing that!

HTML

Open <a href="https://google.com">Google</a>

Output

Open Google


Markdown

Open [Google](https://google.com)

Output

Open Google


Ok, how to use this?

First in [] add the text and in () place the link.
Now you created a link in Markdown.
