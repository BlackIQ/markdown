Learn Markdown.

Markdown is a lightweight markup language for creating formatted text using a plain-text editor. John Gruber and Aaron Swartz created Markdown in 2004 as a markup language that is appealing to human readers in its source code form.

Here you will learn about it more and how to use that.


Headings

What are Headings? The title or the name of a document or even an chapter of a book is a Heading. Now, let's learn how to use them!

In HTML we use H tags. These are Headings. But not md. We use # instead of H tags. Fitst I will show you HTML way and next, Markdown!

HTML

<h1>Heading</h1> <h2>Heading</h2> <h3>Heading</h3> <h4>Heading</h4> <h5>Heading</h5> <h6>Heading</h6>

Output

Heading

Ok, right. Let's go for markdown.

Markdown

# Heading ## Heading ### Heading #### Heading ##### Heading ###### Heading
